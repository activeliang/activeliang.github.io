div 和 span 的区别

  1. div是块级元素，是许多元素的容器。 ==以新行开始或结束。==
  2. span是内联元素，是文本的容器。==不会以新行开始或结束。==

    class 和 id的区别

  3. class 是类名，id 是唯一标识。

p 和 br 的区别

  1. 是段落标识符，会在前后自动加空行。==成对出现==

  2. 是换行标识符，只换行。单独出现。
