每日学习纪录：ORID

这是一个思考框架，利用 ORID 整理今天的思绪。

Objective

关于今天的课程, 你记得什么? 完成了什么?

今天的课程是部署，做购物网站……，记得部署遇到有个文件不一样，于是就想变一个方式再开始……完成了部署的一小部分，完成了购物的后台curd

Reflective

你要如何形容今天的情绪 今天的高峰是什么? 今天的低点是什么?

今天情绪起伏，

高峰：晚上申请留级后，和陈玲敏交流，本来还不太好意思告诉她，交流半天后才知道她也打算留级了…………/捂脸。知道后，心花怒放，有战友了……哈哈，期待后面的协作！

低点：

  • 上午部署阿里去遇到不少坑，注册linedo遇到审核的坑，那时有情绪：为什么国外的网站都这么不人性化，注册什么的好多坑！还不好中国的阿里来的贴心，短信，APP全方位的服务一条龙……
  • 部署阿里云的时候，到后面遇到一个文件跟教材不一样，仔细看上面才发现这个教程是主要为linedo写的，这时我的linedo审核也通过了，就想着，干脆在linedo重来一遍好了……
  • 晚上为了解决一个emmet的自定快捷键，折腾了很久，找到了很多线索，明天继续折腾！

Interpretive

我们今天学到了什么? 今天一个重要的领悟是什么?

  • 今天学到了什么：原来，ubuntu系统是Linux系统，前阵辛妃娣给我试的优麒麟也是这样的系统，跟mac很相像呀，然后，部署阿里云，发现访问很快！不错！
  • 今天一个重要的领悟：没

Decisional

我们会如何用一句话形容今天的工作 有哪些工作需要明天继续努力?

今天的工作，有用功太少。

明天继续努力：把emmet自定义搞定！快快打code!练就无影手。。。。。。
