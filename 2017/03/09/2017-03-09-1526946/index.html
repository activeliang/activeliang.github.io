每日学习纪录：ORID

这是一个思考框架，利用 ORID 整理今天的思绪。

Objective

关于今天的课程, 你记得什么? 完成了什么?

今天下午5点重新开始做rails101, 完成了基础的bootstrap套件，然后，在限制不是admin的用户无法进入后台的时候出bug了，解不开。不是admin可以进入后台，是admin反而无法进入后台。反复检查了代码，找不到原因。

完成了：

一个简单的套件铺设。

晚上看了ihower的课程，记得：

  • 在atom里可以用command+t能快速搜索文件位置 ，我之前在youtube看大神们都是这么干的，大神们都有一个特点：就是打字赶快。
  • 用select2来做下拉选单，可以多选，可以搜索再选。。。
  • Data picker，可以用来先日期的
  • autosize可以根据输入大小改变输入框大小
  • bootcdn可以到找js资源。
  • turbolinks，一个加速的内建gem，有时会变成一个大坑，因为它会让浏览器在第二次后不加载,当你感觉js有灵异现象的时候，可以来检查是不是它出问题了……

Reflective

你要如何形容今天的情绪 今天的高峰是什么? 今天的低点是什么?

今天的情绪有点任性了……

高峰：晚上看ihower直播，看到一个chart.js的套件，可以轻松做出图表，太棒了，后面我的app要能做出很漂亮的图表！

低点：今天一天都在解决一个小问题：如何在emmet里增加一个自定义片段，目的：输入“%”+tab键，能自动补全：<%|%>然后还把光标自动停在中间。在atom里有自定义的功能，可以解决这个问题，但是加载emmet插件后会被覆盖。最好的解决方法是自己增加一个emmet的snippets，然后google了一堆资料，这种感觉就像是警察抓小偷，明明有官方文档说可以做得出来，明明跟着做了，明明找到据说的那个文件了，但就是不出效果！！然后就这样迷失地追踪了一天。到下午4点才暂停，就没人做一个可行的教程出来吗？中午找得快疯掉了……

Interpretive

我们今天学到了什么? 今天一个重要的领悟是什么?

今天学到了，如何在atom里用快捷的找文件的方法。还有一些关于前端的小套路。

今天一个重要的领悟是什么：没有，但有反思：

  • 今天好像又掉进了一个完美的坑，想着能找到那个快捷方式以后打code就完美了……,然后今天一大早就开始了疯狂的google 模式，还发了一个gmail给原作者，希望他可以回复！
  • 在slack问助教，助教还是那句话，新手不要用这些工具，要一个一个地打code。
  • 晚上看ihoower直播，他打code的时候，也没有emmet，<%%>他也没有用什么快捷方式，只是跟着顺序打，没有先头跟尾而已。

Decisional

我们会如何用一句话形容今天的工作 有哪些工作需要明天继续努力?

一句话形容今天就是：白天真的是狗吃屎了，晚上看直播还能学到点东西……

明天：要上班了，白天去找财务问清固定资产的账务有哪些是要复杂性劳动的……

晚上回来好好再跟着打商店的code.
