每日学习纪录：ORID

这是一个思考框架，利用 ORID 整理今天的思绪。

Objective

关于今天的课程, 你记得什么? 完成了什么?

今天的课程：购物网站一些不懂的语句google弄明白，还做了一下bootstrap的基础搭建。

记得：查找的东西都做了logdown笔记，

完成了什么：基本就完成了这些。

Reflective

你要如何形容今天的情绪 今天的高峰是什么? 今天的低点是什么?

今天的情绪：有点儿纠结了，不知道要不要留级，同学都觉得不要留级，但是我那颗心想留级再学一下。

高峰：看rails102,体会到了xdite老师说的：匹配记忆。匹配记忆的过程很爽！

低点：晚上注册linode的时候把用户名打错了，操之过急所致！

Interpretive

我们今天学到了什么? 今天一个重要的领悟是什么?

今天我学到了什么：查了不明白的语句，基本上可以看懂，也体会到了那句话：原来像外星文一样的文字，瞬间就不怕了，就感觉顺很多了……

今天一个重要的领悟是什么：不讨厌==不喜欢，看了xdite老师一些以前的博客，相见恨晚，xdite老师说她后悔4年之才去做自己喜欢的事情（编程），我想我也是四年之后才去做自己的喜欢的事情！

Decisional

我们会如何用一句话形容今天的工作 有哪些工作需要明天继续努力?

反思：做事的时候目标不够明确呀，甚是明确了怎么不会去打开朋友圈，怎么还会效率低呢……目标不够明确。没有游戏化，没有精力管理！

明天要继续努力的：快快把部署的教程给过了……然后申请表给填了……

全栈营购物网站一些不明语法的google结果。
