  2. 中午自己盲打做出来删除购物项，太帅了！！！！心情大好！
  3. 我想到了怎么用rails开发 微信公众号来盘点固定资产了！哈哈！已经有思路了，等待试验中！！！！！！！
  4. 晚上听笑来直播：
    • 什么是执行力：即便做得不好，也会持续去做！
    • 当你身边的人都很厉害的时候，你也会变得很厉害！
    • 到最后，你会发现：学任何困难的事情都一个方法：重复。

每日学习纪录：ORID

这是一个思考框架，利用 ORID 整理今天的思绪。

Objective

关于今天的课程, 你记得什么? 完成了什么?

今天的任务是做购物网站。

还记得：

  • 晚上看了笑来直播，又很多收获，说到这里又涉及到一个执行力的问题了。
  • 今天竟然就把前面四天的事情（做一遍购物网站）给一天做完了。也没有遇到大的BUG。

完成了什么：完成了购物网站一次，看了笑来直播，收获很多。

Reflective

你要如何形容今天的情绪 今天的高峰是什么? 今天的低点是什么?

今天情绪不错。微信扫码功能，也是我想多了，终于想到些眉目了。

高峰：中午想到微信公众号盘点要怎么做。

低点：没有！

Interpretive

我们今天学到了什么? 今天一个重要的领悟是什么?

今天学到了什么

  • 今天做购物打code可以压缩到8个小时了，明天继续！
  • 看直播提取了些东西。请看上面。

今天重要的领悟：

  • 什么是执行力：即便做得不好，也会持续去做！
  • 到最后，学难的东西，方法就一个：重复。
  • 分享一个东西，别人不知道它的好，是不会做的。当别人给你分享东西的时候，要注意到这点！去做，去尝试.

Decisional

我们会如何用一句话形容今天的工作 有哪些工作需要明天继续努力?

今天的打code有进步！把前面4天的东西，今天一天就打完了，还很少出错！哈哈。

明天要继续努力的：再高频地打code,压缩再压缩！
