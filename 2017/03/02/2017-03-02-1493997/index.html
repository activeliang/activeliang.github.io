2017-3-2学习记录ORID

发表于 2017-03-02 |

  1. 图片用了自响应responsive后不居中，只要加一个display:inline;就可以居中了。

  2. 今天的高点：下午和公司IT权哥，YY老师聊rails 的事情，感觉对未来要做出来的事情 很爽，哈哈，YY老师，说我现在是程序员了吗？太炒了吧？哈哈……学！后面有部署的教程，太适合我！。

  3. 高点2： 遇到了一个很好的商店网站：吉盟珠宝，这个网站很适合新手去做！在想要是能早点知道这个网站，跟着做，不比现在第一名的网站差！

  4. bootstrap的清除浮动。遇到清除浮动的语句，会自动重新起一行开始堆叠。<div class="clearfix visible-sm"></div> 这个语句的意思是在sm中执行这个语句。遇到这个语句，不论后面有没有位置，都会另起一行重新开始堆叠。

  5. 今天遇到的一个大坑：解决方法：在container下加多一个row，就搞定啦。这个原先我是知道的，但是没遇过这个坑，没能及时想起来……下次一定能第一时间想起来……。

  6. 记录一个小技巧好工具：codepen。这个工具可以帮你更好的：

    • 集中注意力在关键代码上，避免你的头脑太乱。
    • 能立即显示效果。

每日学习纪录：ORID

这是一个思考框架，利用 ORID 整理今天的思绪。

Objective

关于今天的课程, 你记得什么? 完成了什么?

今天的课程是学习模仿吉盟有购物网站，期间遇到一个大坑，竟也想起解决方法了……

完成了网站首页有三分之二了……

Reflective

你要如何形容今天的情绪 今天的高峰是什么? 今天的低点是什么?

今天的情绪是很嗨的。听着电子音乐，有点节奏的感觉 。

今天的高峰：3个

  • 中午公司IT的权哥，YY老师聊以后做网站的部署问题，YY老师以为我这么快就当止程序员了，太妙了，我也高兴了一下下，心情好……
  • 今天找到一个很好的购物网站：吉盟珠宝。相见恨晚呀。
  • 前两天跟着W3SCHOOL的bootstrap过了一部分教程，现在做页面终于有感觉了……无论是页面分布还是……总之，终于有感觉 了……哈哈
  • 今天跟队友合作很开心，有队友，然后不为啥，只为学习，为了同一个目标，倾心合作，这种感觉太好！太棒了……期待下次还有这种感觉……。

今天的低点：今天的低点就是在遇到以上这些高点之前，我还在为做页面痛苦不已，因为没有东西可以模仿。当你找到东西可以模仿的时候，你就不痛苦了……模仿多了，可能也不需要怎么模仿了……哈哈。

Interpretive

我们今天学到了什么? 今天一个重要的领悟是什么?

今天学到了什么：都在前面列着呢。

今天一个重要的领悟是什么：

  • 有些东西相见恨晚呀！其实之前是见过面的，但是当时没能入耳……错过不知错过……如何避免这种情况？
  • 学东西的时候，最好的就是模仿，最快，学得最快。没知道的时候去创作，会很痛苦，做出来的还不好，会极大地打击你的成就感。！！！切记。

Decisional

我们会如何用一句话形容今天的工作 有哪些工作需要明天继续努力?

太好了……今天这种状态才是好的状态，才是快乐的学习嘛。 是不是在听的电子音乐的节奏有关，太嗨。

明天要继续努力的：快快做页面，产品快快上线，然后把那个教程一贴，拉票……然后留级吧……/捂脸……
