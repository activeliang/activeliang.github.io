mac上一键离线整个网站的方法（以便断网浏览）

发表于 2017-10-18 | 分类于 mac |

写在前面：

最近vpn容易断（大环境所致）。于是思考如何离线浏览一个网站。别说，还真找到了简单粗暴的方法。可以实现：一键离线整个网站（包括图片和视频），以便在火车上，飞机上，断网的情况下浏览网页，尽情学习。

开始:（四步）

  1. 下载安装软件。点击下载

  2. 连接VPN（如果你要离线的网页需要的话。）

  3. 打开软件，输入网址，这里以我经常查阅的 W3school 的 JS 课程 为例。

    ​

  4. 离线选项，点击下载：

    说明：
    • 离线页数限制：
      • 限制页数的大小，根据你要离线的范围决定，最高可以一次离线5000个页面（一个链接就是一个页面）
      • 我们当前要离线的js教程，估计有几千个链接总数。这个无法准确的预知，我们先离线1000个看看，不够再加。
    • 链接层级限制：根据需要设置，如果层级不太明确，请选Unlimited
    • 层级选项：根据字面意思理解。
    • 外站离线层级限制： 下载外站链接的页面时，层级限制。

    其他配置，各位自行研究下。。。
