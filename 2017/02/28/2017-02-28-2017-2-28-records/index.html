每日学习纪录：ORID

这是一个思考框架，利用 ORID 整理今天的思绪。

Objective

关于今天的课程, 你记得什么? 完成了什么?

今天的课程是学习前端知识，学习很多bootstrap，以前不懂的，现在脑子里有些记忆可以匹配了，所以，学得很舒服……完成了很多bootstrap的基础知识。还做了笔记。可以看看同天的bootstrap笔记。

Reflective

你要如何形容今天的情绪 今天的高峰是什么? 今天的低点是什么?

今天的心情总算是好点了，因为今天终于有点儿新知识的收获了，掉坑掉了两天，现在终于好点了……

今天的高峰：早上11点半，终于找到了bootstrap学习的地方，从些套版不再一头雾水。

今天的低点：晚上还是跟看通哥看了一下电视剧。然后，想起队友，觉得自己的这个进度不知道该怎么讲。情况不妙。

Interpretive

我们今天学到了什么? 今天一个重要的领悟是什么?

今天学到了：bootstrap的基础知识。

今天一个重要的领悟：去运动的路上，想起了这两天升级了一个概念，瞬间觉得自己是又重生了：

接受现在不完全的现状，然后要有耐心，每天持续执行力，跟时间做朋友，不断完善……只是时间问题，你会越来越好的……

Decisional

我们会如何用一句话形容今天的工作 有哪些工作需要明天继续努力?

今天的工作终于是收入了一点新知识啦，不错，虽然这些知识，在同学的jblisting大赛他们就学会了……好落后……但是总算是有收获的。

明天要继续努力的：想想自己进度，要如何走接下来的一个星期。重新调整下。
